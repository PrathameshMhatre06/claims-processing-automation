\# Claims Processing Automation



This project demonstrates a rule-based claims processing automation system for a TPA.



\## Features

\- Claim submission API

\- Business rules for auto-approval and manual review

\- PostgreSQL database for persistence

\- Docker and docker-compose setup

\- OpenAPI (Swagger) documentation

\- Postman-tested APIs



\## Tech Stack

\- FastAPI

\- Pydantic

\- PostgreSQL

\- Docker \& Docker Compose



\## How to Run

```bash

docker-compose up --build
